<h2 id="started">🚀 Getting started</h2>

<h3>Prerequisites</h3>

- [NodeJS](https://github.com/)
- [Git 2](https://github.com)

<h3>Cloning</h3>

```bash
git clone https://github.com/andrewsys/sakura-sushi.git
```

<h3>Starting</h3>

```bash
cd sakura-sushi
npm install
npm run dev
```
